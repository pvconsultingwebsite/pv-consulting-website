# Praneet Velaga Consulting LLC - Website

This is a lightweight static site (no backend). It includes:
- Home
- Services
- Dynamics Resources (search + filters)
- Blog (starter list)
- About
- Contact (mailto form)

## Quick publish options

### 1) GitHub Pages (free)
1. Create a repo and upload these files.
2. Settings → Pages → Deploy from branch (main) → /(root)

### 2) Azure Static Web Apps (recommended)
1. Create a repo on GitHub with these files.
2. Azure Portal → Static Web Apps → Create
3. Link GitHub repo and deploy (Azure adds free SSL)

### Customize
- Replace placeholder links (#) in resources.html data with your GitHub links.
- Update email/phone/LinkedIn in contact.html / header.
